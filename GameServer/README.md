# GameServer

### commands:
- .speedup \<float value>
- .spawn \<npcId>
- .itemdrop \<itemId> \<?itemCount> \<?itemPlus>
- .itemget \<itemId> \<?itemCount> \<?itemPlus>
- .search item \<string> (not all results will be displayed due to ingame chat limitation, pagination is needed)
- .levelup \<?level>

? - optional argument
